# **App Name**: ArchitechNexus

## Core Features:

- Professional Hero Section: Displays the developer's main expertise, a brief introduction, and calls-to-action like 'View Projects' and 'Contact Me', complemented by a dynamic background pattern.
- About & Expertise Showcase: Presents the developer's 4.5+ years of backend experience, specializing in Node.js, Microservices, AWS Cloud, REST APIs, and scalable system architecture, with relevant tech stack icons.
- Comprehensive Tech Stack Display: Organizes and displays backend, cloud & DevOps, database, and tool technologies using modern, categorized cards and animated tech stack logos.
- Curated Project Portfolio: Showcases featured projects like 'Qiklet', 'IoT Device Platform', and 'AWS Infrastructure Optimization', with detailed descriptions, tech badges, GitHub links, and architecture diagrams.
- Dynamic Experience Timeline: Visualizes the developer's career journey, highlighting roles, companies, dates, and key achievements in a clean timeline UI.
- Direct Contact & Inquiry Form: Provides easy access to email, LinkedIn, and GitHub, along with a functional contact form for prospective clients or employers to send messages.
- AI-Assisted Project Insight: A tool on project detail pages that allows visitors to ask specific questions about the project's technical aspects or challenges, receiving AI-generated insights or summaries.

## Style Guidelines:

- Dark color scheme with a background of very dark charcoal (HSL: 270, 15%, 10% / #1B151E), providing a professional and tech-inspired foundation.
- Primary accent color: A vibrant purple (HSL: 270, 75%, 60% / #894DCC), used for headings, interactive elements, and subtle gradients to align with the 'purple gradient accents' request.
- Secondary accent color: A bold magenta (HSL: 300, 80%, 65% / #E654D6), creating strong contrast for highlights, hover states, and further emphasizing the modern tech aesthetic.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy, and computerized feel, suitable for prominent display text.
- Body font: 'Inter' (sans-serif) for clean, objective, and highly readable paragraph text across the portfolio.
- Code font: 'Source Code Pro' (monospace) for displaying any code snippets or technical references, ensuring clarity and developer familiarity.
- Minimalistic line-art or glyph-style icons for navigation and general UI elements. Use official, recognizable logos for all displayed tech stacks and tools to maintain professionalism and brand identity.
- Utilize a responsive, grid-based layout for organized content display, especially for projects and tech stack sections. Implement modern UI styles including subtle glassmorphism effects on cards and sections, paired with gentle linear gradients to create depth against the dark theme.
- Leverage Framer Motion for smooth, deliberate scroll-triggered animations (e.g., fade-in, slide-up effects for content), interactive hover states on cards and buttons, seamless page transitions, and subtle background animations like a glowing grid or flowing code pattern.
